# Deployment Notes

This is a static web app. You can host it anywhere that supports HTML, CSS, and JavaScript.

## Option 1: GitHub Pages
1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `app.js`.
3. Go to Settings > Pages.
4. Deploy from the main branch.
5. Use the generated GitHub Pages URL in the Skip application.

## Option 2: Netlify
1. Go to Netlify.
2. Drag and drop this folder.
3. Copy the live site URL.

## Option 3: Vercel
1. Create a new Vercel project.
2. Import the GitHub repository or upload the static files.
3. Use the generated deployment URL.

## Option 4: Any Static Hosting Provider
Upload these files to the public web root:
- index.html
- styles.css
- app.js

## Demo checklist before submitting
- Open the live link.
- Load a sample business.
- Generate the report.
- Test Print / Save as PDF.
- Test Download Text Report.
- Record the Loom walkthrough.
