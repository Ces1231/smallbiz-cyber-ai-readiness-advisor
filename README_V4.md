# SmallBiz Cyber & AI Readiness Advisor V4

## New V4 Business Owner Action Center

This version adds practical business-value features:

- Priority Action Checklist
  - Do this first
  - Do this next
  - Do this later
- Estimated Cost Level
  - Free
  - Low cost
  - Medium cost
  - Requires professional help
- Tool Recommendations
  - Password manager
  - Backup tool
  - AI scheduling tool
  - CRM
  - Accounting/invoicing tool
- Document Generator
  - Cybersecurity policy
  - AI usage policy
  - Business continuity checklist
  - Grant readiness checklist
- Monthly Reassessment
  - Save baseline score
  - Recheck in 30 days
  - Show improvement
- Business Risk Register
  - Risk
  - Impact
  - Priority
  - Recommended fix

## Static Hosting

Upload these files to Serverbyt-style static hosting:

- index.html
- styles.css
- app.js

The app is static and does not require Node.js or a backend.

Note: Chart.js loads from a CDN. If your host blocks CDN scripts, replace the CDN with a local Chart.js file.
