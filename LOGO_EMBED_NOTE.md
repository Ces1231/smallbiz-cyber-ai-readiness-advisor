# Logo Embed Note

The Champtron Systems LLC logo is embedded directly inside `index.html` as a Base64 data URI.

This means the logo does not depend on a separate image path, filename, or upload location.

For Serverbyt hosting, upload:
- index.html
- styles.css
- app.js

Optional:
- champtron-logo-optimized.jpg
- README.md
- deployment notes

The embedded logo should display even if image files are not uploaded.
